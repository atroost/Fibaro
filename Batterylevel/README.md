# Batterylevel
Batterylevel sends an email if a device needs a replacement of the battery.

## Requirements
- Depends on sendMessage scene

## Installation
##### In HC2:
- Go the Scenes
- Add scene
- Add scene in lua
- Paste the lua code
- Give the scene a name
- Assign it to a room
- Save
- Add icon to scene

That is it!
